* Regole generali: 
   * Svolgi TUTTI gli esercizi qui di seguito in una repository di github di cui ci invierai poi il link.
   * Ricordati che l'obiettivo principale è scrivere codice che funziona, certo, ma anche che altri esseri umani possono facilmente comprendere. Dove credi sia necessario, inserisci commenti.
   * Per completare gli esercizi, puoi utilizzare un linguaggio a scelta fra Javascript, PHP o Python.
   * Se hai domande, scrivi a board@mumbleideas.it. Ricordati di formulare le tue domande in modo chiaro, riportando esempi chiari. In linea di massima, ti consigliamo di non fare domande, leggere bene il testo dell'esercizio e fidarti del tuo intuito.
   * Per gli esercizi in cui esistono diversi livelli, fai del tuo meglio per risolverli tutti. Non cancellare o sovrascrivere il codice del livello precedente; commentalo e lascialo nello stesso file.
* Quick quiz
   * Analizza palindromi: crea una funzione che ritorni true in caso la stringa passata come argument sia palindroma o false in caso non lo fosse.
      * Livello 1: rileva il palindromo soltanto se la stringa non contiene spazi o altri simboli e le lettere sono tutte minuscole
      * Test da superare:
palindromo('radar') // => true
palindromo('sonar') // => false

      * Livello 2: rileva il palindromo anche in caso la stringa contenga spazi e contenga lettere maiuscole
      * Test da superare:
palindromo("Ena sellava cavalle sane") // => true
palindromo("A caval donato") // => false


   * Conta le vocali:
      * Livello 1: Scrivi una funzione che conti il numero totale di vocali contenute in una stringa
      * Test da superare:
contaVocali('cammello') // => 3
contaVocali('arcobaleno') // => 5

      * Livello 2: Scrivi una funzione che conti il numero di vocali in una stringa e ritorni il risultato sotto forma di oggetto contenente tutte le vocali e il rispettivo conteggio
      * Test da superare:
contaVocali('Come vola sto cammello') 
// => {a: 2, e: 2, i: 0, o: 4, u: 0}
